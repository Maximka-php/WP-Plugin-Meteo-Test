Тестовый плагин для вывода прогноза погоды на страницах.

2 варианта работы: прогноз выводится индивидуальный для каждого пользователя(координаты определяются по IP);
                   прогноз выводится для определенного места(указывается широта и долгота);
Вариант работы можно выбрать на странице настроек в админке ;

Для работы необходимо получить токен от Яндекса(есть бесплатные планы с ограничениями)
    https://yandex.ru/dev/weather/doc/dg/concepts/pricing.html ;

Вывод прогноза погоды осуществляется путем вставки шорткода [yandex_meteo] ;
